# Sobre o Projeto

Sistema de Reserva de Espaços Públicos desenvolvido em PHP para gerenciamento de espaços, usuários, eventos e reservas. O sistema permite realizar operações de cadastro, consulta, alteração e exclusão (CRUD), além de registrar reservas relacionando usuários, espaços e eventos com data e horário. Os dados são armazenados em arquivos JSON e a interface foi construída utilizando Bootstrap para proporcionar uma navegação simples e intuitiva.
